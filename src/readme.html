Инструкция по использованию 3DViewer v1.0

Установка:

  1. Вызвать цель make install из папки src.
  2. Дождаться установки *.app.

Сборка в архив:

  1. Вызвать цель make dist из папки src.
  2. Архив собирается в папку src.

Удаление:

  1. Вызвать цель make uninstall из папки src.
  2. Дождаться удаления  *.app.

Запуск и использование:

  1. Для запуска приложения открыть исполняемый файл open ./build/3DViewer.app/Contents/MacOS/3DViewer, либо через терминал командой "open build/3DViewer.app"
  2. Программа загружает файлы в формате *.obj.
  3. После загрузки объектного файла доступен весь функционал в графическом интерфейсе, перемещение по осям возможно при помощи соответствующих слайдеров на цифровой панели программы.
  4. Для выбора необходимого цветового решения необходимо использовать кнопки "color verticies", "color edges", "Background color". При помощи кнопки "Reset values" можно сбросить настройки пользователя на настройки установленные разработчиками по умолчанию. Для сохранения в формате bmp, jpg, gif, использовать кнопки "bmp", "jpg" и "gif" соответственно.
  • 3DViewer v1.0 поддерживает следующие действия:
    • Перемещение и вращение:
      1. Перемещение по оси X: "Translate X: от -180 до +180"
      2. Перемещение по оси Y: "Translate Y: от -180 до +180"
      3. Перемещение по оси Z:"Translate Z: от -180 до +180"
      4. Вращение по оси X: "Rotate X: от -180 до +180"
      5. Вращение по оси Y: "Rotate Y: от -180 до +180"
      6. Вращение по оси Z: "Rotate Z: от -180 до +180"
      7. Приближение: "Scale от 1 - до 100"
    • Опции:
      1. Тип проекции: "Central" - центральная, "parallel" - параллельная
      2. Тип вершин: "Disable" - невидимые, "Circle" - шарообразные , "Square" - квадратные
      3. Тип граней: "Solid - прямые, "Dashed" - пунктирные
      4. Толщина граней: "Thickness"
      5. Цвет вершин: "color verticies"
      6. Цвет граней: "color edges"
      7. Цвет заднего фона: "Background color"
      8. Сброс значений на заводские настройки: "Reset values"
      9. Сохранение значений в формате: bmp "bmp"
      10. Сохранение значений в формате: jpg "jpg"
      11. Сохранение значений в формате: gif "gif"
